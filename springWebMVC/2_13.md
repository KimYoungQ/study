HTTP 메시지 컨버터 2부 - JSON
===

#### 스프링 부트를 사용하지 않는 경우
  ● 사용하고 싶은 JSON 라이브러리를 의존성으로 추가  
  ● GSON    
  ● JacksonJSON 
  ● JacksonJSON 2 

<br/>

#### 스프링 부트를 사용하는 경우  
  ● 기본적으로 JacksonJSON 2가 의존성에 들어있다. 
  ● 즉, JSON용 HTTP 메시지 컨버터가 기본으로 등록되어 있다.  
  
<br/>
